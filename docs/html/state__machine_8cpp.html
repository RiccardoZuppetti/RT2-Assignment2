Description:
This node is a server for the user interface. It will receive commands from the user interface. It can elaborate new random target from the position_server. It can start or stop the robot motion with the go_to_point action.

This function initializes the ros node, the server, the clients. Then if ros is running it checks the global variable start. If start is true and the robot is not already moving a new goal is set. If the robot is already moving then it is checked if the goal is reached. If the global variable start is false and the robot is moving then all the goals will be canceled.
